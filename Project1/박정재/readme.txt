1. 게임 실행 시 필요한 image 폴더 내 이미지와 audio 폴더 내 오디오를 꼭 내려받아야 한다.

공략법
* 효과음이 있으니 소리를 키고 플레이 해주시길 바랍니다.
* 검은색 화살표를 클릭하면 이동이 가능합니다.
* '20151870_myGame.js'를 불러오면 됩니다.

1. 정신을 차려보니 수갑이 채워진 채 감옥에 갇혀 있다.
2. 감옥 바깥에 누군가 실수로 떨어뜨리고 간 열쇠가 보인다. 수갑을 열 수 있을 것 같다. 간신히 손이 닿는다.
3. 바깥으로 향한 화살표 클릭 시, 감옥 바깥에 키패드와 호수가 희미하게 보인다.
	- 열쇠를 이용하여 수갑을 풀었을 시, 키패드에 손이 닿는다.
4. 감옥 내부 벽에 적혀 있는 식과 호수가 철창 문을 여는데 관련이 있다.
	- 벽에 있는 식의 왼쪽 숫자를 소인수분해하면 오른쪽 숫자가 나온다.
	- 플레이어가 갇혀있는 호수인 302를 소인수분해하면 2151이다.
	- 키패드에 2151 입력 시, 철창 문이 열린다.
5. 키패드에 2151 입력 시, 철창 문이 열린다.
6. 화살표를 클릭하여 탈출한다.
